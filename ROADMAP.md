# Roadmap

This document outlines the planned features and enhancements for Appliqué.

## Project Status

Appliqué is currently in active development. The core features are functional, but the project is being actively expanded with new capabilities and improvements.

## Future Considerations

The following features are under consideration for future releases:

### Language Support
- Expand beyond EN/DE to support additional languages
- Locale-specific formatting (dates, addresses, etc.)
- Language detection from job postings

### CV Tailoring
- Skill matching and highlighting
- Experience reordering based on relevance

### Enhanced Agent Workflows
- Multi-step verification and fact-checking
- Tone and style adaptation based on company culture

### Integration Features
- Direct job board integration (LinkedIn, Indeed, etc.)

### Enhanced Observability
- Application analytics dashboard
- LLM usage tracking and cost optimization

### Rendering Backend Support
- Add [Typst](https://typst.app/) as an alternative to LaTeX
